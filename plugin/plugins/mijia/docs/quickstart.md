# 米家智能家居快速开始

## 简介

米家智能家居插件可以让你通过 AI 伙伴控制小米/米家智能家居设备，包括灯光、空调、插座、传感器等。

## 快速上手

### 第一步：登录米家账号

在插件面板中点击**扫码登录**按钮，使用米家APP扫描二维码完成授权。

你也可以直接对 AI 说：

> "帮我登录米家账号"

### 第二步：获取设备列表

登录成功后，点击**刷新设备**按钮，插件会自动获取你账号下的所有米家设备。

### 第三步：控制设备

在"智能控制"区域输入控制命令：

- **开灯/关灯**：`打开客厅灯`、`关闭卧室灯`
- **空调控制**：`打开空调`、`关闭空调`
- **插座控制**：`打开插座`、`关闭卧室插座`

你也可以直接对 AI 说：

> "帮我打开客厅灯"
> "关闭卧室空调"

### 第四步：执行智能场景

在"智能场景"区域，点击场景旁边的**执行**按钮，触发米家APP中预设的智能场景。

## 功能概览

| 功能 | 说明 |
|------|------|
| 二维码登录 | 使用米家APP扫码登录，安全便捷 |
| 设备控制 | 支持开关、亮度、温度等属性控制 |
| 智能场景 | 执行米家APP中预设的自动化场景 |
| 设备别名 | 自定义设备别名，方便语音控制 |
| 自然语言 | 支持"打开客厅灯"等自然语言命令 |

## 入口说明

| 入口 | 说明 |
|------|------|
| `start_qrcode_login` | 开始二维码登录流程 |
| `logout` | 登出并清除本地数据 |
| `list_devices` | 获取设备列表（支持缓存） |
| `smart_control` | 自然语言控制设备 |
| `execute_scene` | 执行智能场景 |
| `control_device` | 精确控制设备属性 |
| `query_device_state` | 查询设备当前状态 |

## 常见问题

**Q：登录失败怎么办？**
A：确保米家APP已安装并登录，扫描二维码时请使用米家APP而非微信。

**Q：设备不在线？**
A：检查设备是否通电，网络是否正常。部分设备需要网关才能连接。

**Q：控制命令不生效？**
A：确保设备名称正确，或使用设备别名。可以先用"查找设备"确认设备存在。

**Q：如何设置设备别名？**
A：在设备列表中找到设备，点击编辑别名。支持多个别名用逗号分隔。

## 依赖

- Python 3.11+
- `httpx`（HTTP 请求）
- `pydantic`（数据验证）
- `qrcode`（二维码生成）
- `tenacity`（重试机制）
- `cachetools`（缓存管理）

以上依赖均由 NEKO 内置，无需额外安装。
