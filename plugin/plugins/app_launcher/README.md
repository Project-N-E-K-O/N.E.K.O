# 🚀 应用启动器 (App Launcher) 插件

> 让猫娘帮你管理电脑上的软件，一键启动，还能设置开机自启！

## ✨ 功能特性

### 🎯 核心功能
- **软件管理**：添加、删除、编辑、查找电脑上的软件
- **一键启动**：猫娘可以帮你打开任何已注册的软件
- **开机自启**：设置软件开机自动启动（仅支持 Windows）
- **路径管理**：支持 .exe、.lnk 快捷方式、.bat 脚本等多种类型

### 🎨 界面特色
- **清新明亮**：诗意浪漫的渐变色设计
- **响应式布局**：完美适配手机和电脑
- **最近使用**：快速访问常用软件
- **搜索过滤**：快速找到目标软件

### 🌍 多语言支持
- 🇨🇳 中文（简体）
- 🇹🇼 中文（繁體）
- 🇺🇸 English
- 🇯🇵 日本語

## 📖 使用教程

### 🎮 基础操作

#### 1. 添加软件

**方式一：通过猫娘语音**

```text
你：猫娘，帮我添加 Steam
猫娘：好的主人，请告诉我 Steam 的安装路径~
你：C:\Program Files (x86)\Steam\Steam.exe
猫娘：已成功添加！我记住了，以后你说"打开 Steam"我就能帮你启动啦~
```

**方式二：通过 Web UI**

1. 点击「➕ 添加软件」按钮
2. 填写软件名称（如：Steam）
3. 填写文件路径（如：`C:\Program Files (x86)\Steam\Steam.exe`）
4. 可选：添加别名（如：steam, 游戏平台）
5. 可选：添加描述
6. 点击保存

#### 2. 启动软件

**语音方式：**
```text
你：猫娘，打开 Steam
猫娘：好的主人，正在为你启动 Steam~ ✨
```

**UI 方式：**
- 在软件列表中找到目标软件
- 点击右侧的 ▶️ 按钮

#### 3. 设置开机自启

**语音方式：**
```text
你：猫娘，让 Steam 开机自启
猫娘：好的主人，已为 Steam 设置开机自启~ 下次开机它会自动启动哦！
```

**UI 方式：**
- 在软件列表中找到目标软件
- 点击右侧的开关按钮（仅 .exe 文件可用）

### 🔍 高级功能

#### 1. 使用别名

添加软件时可以设置别名，方便猫娘识别：

```text
你：猫娘，帮我添加网易云音乐，别名是"云音乐"和"音乐"
猫娘：好的！以后你说"打开云音乐"或"打开音乐"我都能识别~
```

#### 2. 搜索软件

在 Web UI 顶部的搜索框中输入关键词，可以快速过滤软件列表。

#### 3. 查看最近使用

Web UI 会显示最近启动过的软件，点击即可快速再次启动。

#### 4. 导出配置

点击「📤 导出配置」按钮，可以将所有软件配置导出为 JSON 文件，方便备份或迁移到其他电脑。

## ⚙️ 配置说明

### 插件配置文件

插件配置位于 `plugin.toml`：

```toml
[plugin]
id = "app_launcher"
name = "应用启动器"
version = "1.3.0"

[plugin.i18n]
default_locale = "zh-CN"
locales = ["zh-CN", "zh-TW", "en", "ja", "ko", "ru", "pt", "es"]
```

### 数据存储

插件数据存储在 N.E.K.O 的 PluginStore 中：
- `app_launcher_apps`：软件列表
- `app_launcher_categories`：分类设置
- `app_launcher_recent`：最近使用记录

## 🔧 API 接口

### 列出所有软件
```python
plugin.call("list_apps")
# 返回: { "apps": [...], "count": 10 }
```

### 添加软件
```python
plugin.call("add_app", {
    "name": "Steam",
    "path": "C:\\Program Files (x86)\\Steam\\Steam.exe",
    "aliases": ["steam", "游戏平台"],
    "description": "游戏平台"
})
```

### 启动软件
```python
plugin.call("launch_app", {"app_id": "xxx"})
# 或
plugin.call("launch_app", {"name": "Steam"})
```

### 设置开机自启
```python
plugin.call("set_autostart", {
    "app_id": "xxx",
    "enabled": true
})
```

### 获取最近使用
```python
plugin.call("get_recent_apps")
# 返回: { "recent_apps": [...], "count": 5 }
```

### 导出配置
```python
plugin.call("export_apps")
# 返回: { "apps": [...], "export_data": {...} }
```

## ❓ 常见问题

### Q: 为什么开机自启设置失败？
A: 开机自启功能仅支持 Windows 系统，且需要 .exe 类型的可执行文件。如果遇到权限问题，请以管理员身份运行 N.E.K.O。

### Q: 快捷方式（.lnk）能设置开机自启吗？
A: 不能。快捷方式可以添加和启动，但开机自启只支持 .exe 文件。

### Q: 如何在多台电脑间同步配置？
A: 使用「导出配置」功能将配置导出为 JSON 文件，然后在其他电脑上导入即可。

### Q: 支持中文路径吗？
A: 支持！插件会自动处理路径编码。

### Q: 如何删除最近使用记录？
A: 最近使用记录会自动保留最近 10 条，超过后会自动清理。
