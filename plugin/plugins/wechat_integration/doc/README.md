# 微信集成

通过 OpenClaw API 连接微信个人号，支持扫码登录和凭证管理。

## 快速开始

1. **开始扫码登录**：在插件 UI 页面中点击「开始扫码登录」按钮
2. **微信扫码**：使用手机微信扫描页面上的二维码
3. **确认登录**：在手机上点击「确认登录」
4. **完成**：页面会自动更新并保存凭证，显示账号信息

## 配置说明

- **API 基础地址**：微信 OpenClaw API 的基础 URL，默认为 `https://ilinkai.weixin.qq.com`，一般不需修改
- **Bot 类型**：微信 Bot 类型编号，默认为 `3`

## 功能说明

- **扫码登录**：支持微信个人号二维码扫码登录，凭证自动保存
- **消息监听**：支持 Start/Stop 控件控制消息监控启停
- **自动回复**：接收消息后通过 LLM 生成智能回复，支持对话历史记录
- **Memory Server 集成**：将对话持久化到 Memory Server，重启后恢复上下文

## 注意事项

- 二维码有效期为约 5 分钟，过期后需点击「刷新二维码」
- 登录凭证会自动保存，重启后无需重新登录
- 微信集成为主人专用（master-only），不支持多用户多会话
