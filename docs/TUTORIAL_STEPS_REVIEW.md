## 审查说明

1. **如何删除步骤**：在 `static/universal-tutorial-manager.js` 文件中找到对应页面的配置，删除不需要的步骤对象即可。

2. **步骤编号**：删除步骤后，系统会自动重新编号，无需手动调整。

3. **建议保留的步骤**：
   - 欢迎介绍（第一步）
   - 核心功能介绍
   - 完成提示（最后一步）

4. **可以考虑删除的步骤**：
   - 过于简单的功能（如关闭按钮）
   - 用户可以自行发现的功能
   - 不常用的高级功能
