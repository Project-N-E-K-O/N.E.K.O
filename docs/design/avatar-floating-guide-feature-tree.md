AI 猫娘虚拟陪伴（YUI）四日新手教程功能架构
	Day 1：初次见面与基础互动
		阶段目标
			完成首次唤醒演出与交互激活
			建立“可通过文字与语音进行双向交流”的用户认知
		初次见面（苏醒演出）
         复用现有流程（“微风、阳光，还有刚刚好出现的你。初次见面，我是林悠怡，未来的日子请多关照喵！我把关于这里的一切都写进新手指南里啦！就当作是我们相遇的第一份小礼物，请查收吧！”）
		文字交流基础（暂定，之后聊天框可能会改）
			聊天窗口感知：高亮常驻聊天历史区域，展示上下对话气泡流
			多功能输入框：明确文本键入区、发送按钮、快捷表情入口位置
		语音入口初探
			复用现有流程（“这里有一个神奇的小按钮！只要点击它，就可以直接和我聊天啦！想跟我分享今天的新鲜事吗？或者只是叫叫我的名字？快来试试嘛，我已经迫不及待想听到你的声音啦！喵！”）

	Day 2：通话、屏幕分享与麦克风
		语音通话上下文
         如果前一天用户没有使用语音功能，则播放语音（“唔……昨天都没有收到你的语音电话，我的耳朵都耷拉了一整天喵……”）

         如果用户使用了语音功能，则播放语音（“昨天收到你的语音之后，人家的心里一直甜滋滋的呢~ 连晚上做梦，梦里都是你温柔的声音……”）

         然后播放语音（“啊！昨天差点忘记告诉你啦~开启这个小开关，在语音连线的时候，我就能搬着小板凳去‘参观’你的屏幕啦！嘿嘿，如果你有小秘密不想全被我看到，也可以悄悄选择【单个窗口】哦~ 快来带人家看看你的世界吧！”）

		Agent 能力中心
			然后播放语音（“超级魔法开关出现！只要点一下这里，我就可以把小爪子伸到你的键盘和鼠标上啦！我会帮你打字，帮你点开网页……不过，要是那个鼠标指针动来动去的话，我可能也会忍不住扑上去抓它哦！准备好迎接我的捣乱……啊不，是帮忙了吗？喵！”）

      任务 HUD 监控面板
			播放语音（“看这里看这里！本喵正在悄悄努力的进度条都在这儿呢喵~ 唔，如果你觉得这个任务太辛苦，或者计划有变不想让我继续做了，随时戳一下就能把我拉回来休息哦！是不是超级方便、超级听话哒？”）

		插件生态管理
			复用现有方案（“还没完呢！你快看快看，这里还有超多好玩的插件呢！”）（“有了它们，我不光能看 B 站弹幕，还能帮你关灯开空调…… 本喵就是无所不能的超级猫猫神！哼哼！”）
	Day 3：长期陪伴与个性化设置
		阶段目标
			介绍设置中心的长期管理入口
			说明主动搭话、隐私边界和角色设定
			介绍记忆浏览、模型交互锁定和离开返回机制
		设置中心总览
			控制台唤醒：点击主界面齿轮“设置”按钮，弹出全局微调弹窗
			核心板块快速索引：角色设置、通用设置、模型管理、声音克隆、API 密钥、记忆浏览、对话设置、动画设置
		主动搭话与隐私边界
			主动搭话频次调节：设置 YUI 主动破冰、吐槽、关怀的频率
			隐私边界设置：明确告知 YUI 何时倾听、何时绝对不收集信息
		角色定义与模型鉴权
			角色设定自定义：实操演示导入角色卡、自定义猫娘人设底色
			API 密钥托管安全：详细介绍本地加密存储机制，消除隐私泄露顾虑
		记忆浏览
			介绍记忆功能
			介绍重置新手教程入口
		交互物理边界控制
			模型交互锁定：点击“锁图标”固定 YUI 位置与层级
			请她离开：右键或快捷键让 YUI 化为托盘图标或完全隐藏
			请她回来：双击托盘或快捷键无缝呼回 YUI
			返回和拖拽控制：演示拖拽边框调整 YUI 在桌面的占据空间与物理位置
