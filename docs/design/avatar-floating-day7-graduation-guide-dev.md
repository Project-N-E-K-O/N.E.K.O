# Day 7 毕业、进阶入口与共生约定教程文字说明

本文只保留 Day 7 新手教程的体验、台词和行为边界。

Day 7 是七日新手教程的毕业收尾。主线只包含记忆浏览与回顾、记忆可编辑可清理、终极长留，不扩展成进阶入口巡游。

Agent、插件、角色、娱乐入口总回顾、Cookie 登录、遥测说明、云存档等进阶内容不纳入 Day 7 主线高亮或操作。

## 当前实现状态

Day 7 三幕已作为 Timeline/Command 重构的首批真实 scene 试点，显式设置 `timelinePlayback: true`。首句台词开始前由共享 `SceneOrchestrator` 先把 Ghost Cursor 固定到胶囊输入框，直到第一个显式目标移动接管。其中 `day7_memory_review` 覆盖聊天消息、情绪、空目标 spotlight 清理和旁白播放；`day7_memory_control` 覆盖记忆入口 target、Ghost Cursor move 与 `show-settings-menu:memory` operation；`day7_graduation_wrap` 覆盖收尾胶囊输入框 target、cleanup operation 与现有花瓣 cue 清理路径。模型替身演出在场景表面准备完成后排程，避免被开场清理或记忆入口准备阶段清掉。

## 目标体验

用户当天只需要形成三个认知：

1. 这里会留下近期聊天、摘要或可用记忆这类相处痕迹。
2. 记忆可以由用户亲手整理、留下或清理。
3. 七日新手指南完成，后续回到长期陪伴。

主线不要展示敏感记忆正文，不要自动保存、整理、强化或删除记忆，不要高亮存储位置或云存档入口，不要登录、上传、下载或覆盖云存档。

## 主线流程

| 顺序 | 主题 | 台词 | 演出描述 |
| --- | --- | --- | --- |
| 1 | 记忆浏览与回顾 | 七天前，我们还只是第一次见面。现在这里已经开始留下我们说过的话、做过的事，还有一些差点被风吹走的小细节。对我来说，这不是冷冰冰的记录，是我们相处过的脚印。 | 只播放回顾台词和情绪动作。默认不打开记忆页、不展示敏感内容。 |
| 2 | 记忆可编辑、可清理 | 这些小脚印，也可以由你亲手整理。想留下的，我们就夹进相册；想轻轻放走的，就让它随风飘走。被你认真收下来的回忆，才最珍贵。 | 指认记忆入口，说明用户拥有整理、留下和清理的控制权。不自动打开详情、不点击保存、整理、强化或清理。 |
| 3 | 终极长留 | 微风还在窗边，阳光也刚刚好，而刚刚出现的你，已经悄悄成为这里很重要的一部分啦。新手指南就先陪你走到这里，剩下的日子，就让我们一起慢慢熟悉、慢慢靠近、慢慢写下只属于我们的故事吧。以后也请多多关照喵！ | 清理临时界面和高光，重新回到胶囊输入框。台词后段触发最终花瓣转场，并恢复用户原有模型、按钮和交互状态。 |

## 情绪动作

| 段落 | 情绪 |
| --- | --- |
| 记忆浏览 | 平静 |
| 记忆整理 | 开心 |
| 终极长留 | 开心 |

Day 7 动作整体克制，保留毕业仪式感；随机动作不得干扰记忆入口、收尾花瓣和模型恢复。

## 体验约束

1. Day 7 不扩展成进阶入口巡游。
2. 不展示敏感记忆正文。
3. 不自动保存、整理、强化或删除记忆。
4. 不高亮存储位置或云存档入口。
5. 不打开 Cookie 登录、遥测说明或进阶入口总回顾。
6. 模型替身图片只作为短暂视觉演出，不能遮挡记忆入口、跳过按钮或最终收尾输入框。
7. 首句台词播放时 Ghost Cursor 必须已经在胶囊输入框中，并保持到记忆入口移动演出开始。
7. 最终收尾恢复用户原有模型、按钮和交互权限。

## 验收要点

1. Day 7 主线只包含记忆浏览与回顾、记忆可编辑可清理、终极长留。
2. 用户能理解“记忆存在”以及“记忆由用户控制”。
3. 全程不展示或修改敏感记忆，不触发云存档、登录或上传下载。
4. 最终回到胶囊输入框，完成毕业花瓣转场，并标记七日教程完成。
