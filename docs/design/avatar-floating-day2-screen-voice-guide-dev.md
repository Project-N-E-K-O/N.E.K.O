# Day 2 个性化、声音与主动搭话教程文字说明

本文只保留 Day 2 新手教程的体验、台词和行为边界。

Day 2 承接第一天的聊天体验，重点让用户知道：可以继续用声音和悠怡相处，可以进入设置描绘更贴近自己的悠怡，也可以找到主动搭话入口。屏幕分享入口不在 Day 2 主线中重复演示。

## 当前实现状态

Day 2 round 已全部接入共享 Timeline/Command 播放路径，覆盖聊天消息、情绪、高光、Ghost Cursor move/click、cleanup operation 和最终花瓣 cue。首句台词开始前由共享 `SceneOrchestrator` 先把 Ghost Cursor 固定到胶囊输入框，直到第一个显式目标移动接管。首句的 `voiceUsed` 分支只读取 Day 1 已记录的语音/会话使用状态，不在 Day 2 现场要求用户重新授权或重新选择输入方式。`day2_personalization_space` 复用旧 `day2-open-settings-personalization` operation，在 Ghost Cursor click 开始时打开设置面板；PC 端全局 overlay 的 Ghost Cursor 使用 46px 猫爪图像中心锚点，且 Timeline 发起的 Ghost Cursor move 使用精确 `durationMs`，避免短移动被全局慢速补偿拉长后与 click/open-settings 节奏错拍。本句移动到设置按钮的时长显式设为 `1480ms`，确保移动足够柔和，并在 click 触发前猫爪中心对齐设置按钮中心。`day2_personalization_detail` 通过显式 `timeline[]` 调用 `settingsTour.play`，旁白、角色侧栏巡游、侧栏关闭和高光清理由 `SettingsTourFlow` 继续控制，且 `timelineAudio: false` 避免重复旁白。`day2_proactive_chat` 使用显式 timeline，在旁白结束后通过 `settingsPanel.close` 关闭临时打开的设置面板并折叠侧栏，不隐藏 Ghost Cursor。模型替身演出在场景表面准备完成后排程，避免被开场清理或设置面板准备阶段清掉。

## 主线流程

| 顺序 | 台词 | 演出描述 |
| --- | --- | --- |
| 1 | 默认：昨天你一直在噼里啪啦打字，我还没听过你说话呢。今天如果愿意，就轻轻叫我一声吧。一句就好，让我把文字背后的你也认识一点点。声音分支：嘿嘿，昨天听到你的声音之后，人家就悄悄把你的语气记在心里啦！今天如果方便的话，也要继续跟人家说话哦~ 虽然打字也可以啦，但只要能听到你的声音，我的尾巴就会开心得一直摇个不停呢，喵呜~ | 播放期间高亮聊天窗，Ghost Cursor 停在聊天区域附近。若用户第一天用过声音，使用声音分支承接；否则使用默认承接。 |
| 2 | 在这个只属于我们的小空间里，你可以由着自己的心意，慢慢描绘出最希望能一直陪着你的那个我。 | Ghost Cursor 从聊天窗以约 1.5 秒的柔和节奏移动到设置入口，猫爪中心对齐设置按钮中心，并演示打开设置面板。 |
| 3 | 不管是说话的温度、相处的小脾气，还是我每天那些细腻的小心思，都可以一点一点调成你喜欢的样子。 | 高亮角色设置区域，Ghost Cursor 指认侧边栏内容。只展示可调整范围，不修改或保存任何配置。 |
| 4 | 这个小按钮也很重要哦，只要你轻轻点一下，我就能在合适的时候跑过去找你啦。 | 高亮主动搭话入口，Ghost Cursor 移动过去并停留。只说明入口，不打开深层设置、不改变开关状态。 |
| 5 | 今天的教程到这里就结束了呢。 | 关闭临时设置界面，重新高亮胶囊输入框，Ghost Cursor 回到输入框附近。 |
| 6 | 其实只要能这样陪着你，听听你的声音，或者静静看着你分享的画面，我就已经觉得很幸福了。 | 继续保持收尾状态，不再触发新的功能演示。 |
| 7 | 我们不需要着急，每天都多了解彼此一点点就好。今天接下来的时间，你想让我陪你做点什么呢？ | 在台词后段隐藏 Ghost Cursor、清理高光，并用花瓣效果结束 Day 2。 |

## 体验约束

1. Day 2 不包含屏幕分享主线。
2. 首句只做承接，不要求用户重新选择“说话”或“打字”；声音分支由 Day 1 的语音使用状态驱动。
3. 设置入口、角色设置区域和主动搭话入口都只做介绍，不保存配置。
4. 主动搭话只展示入口，不触发主动搭话。
5. 收尾前关闭临时设置界面，回到干净的聊天输入状态。
6. 模型替身图片只作为短暂视觉演出，不能遮挡设置入口、主动搭话入口、跳过按钮或收尾输入框。
7. 首句台词播放时 Ghost Cursor 必须已经在胶囊输入框中，并保持到设置入口移动演出开始。

## 验收要点

1. Day 2 主线顺序与本文一致。
2. 默认承接和声音承接两种台词语义保持不变。
3. 用户能理解“个性化设置”和“主动搭话入口”在哪里。
4. 全程不触发屏幕分享、主动搭话或配置保存。
5. 最后回到胶囊输入框，并完成花瓣收尾。
6. PC 端播放 `day2_personalization_space` 时，Ghost Cursor 不能相对设置按钮中心出现偏移；应以猫爪图像中心对齐按钮中心，并在 click/open-settings 触发前完成移动。
